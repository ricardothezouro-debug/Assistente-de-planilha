Política de privacidade

Dados que o app guarda

O app guarda os dados necessários para funcionar: email de login, nome de exibição, categorias, lançamentos financeiros, status de pagamento, convites e registros de auditoria.

Como os dados sao usados

Os dados sao usados para mostrar sua planilha financeira, separar as contas por usuario, permitir backup administrativo e proteger o acesso ao sistema.

Login e infraestrutura

A autenticação é feita pelo Supabase, incluindo login por email/senha e Google. O site é hospedado no Cloudflare Pages e os dados ficam no banco Supabase do projeto.

Admin e auditoria

Contas administradoras podem ver usuários cadastrados, desativar acessos, gerar convites, baixar backup manual e consultar logs de acoes importantes.

Remocao ou bloqueio

Por enquanto, o painel admin desativa contas em vez de apagar permanentemente os dados. Isso preserva historico e evita perda acidental.

Contato

Para pedir ajuste, bloqueio ou remocao futura de dados, fale com o administrador do app.
